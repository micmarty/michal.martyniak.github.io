Recent Major Projects

RustOS

A proof-of-concept operating system written in awesome Rust language + some assembly code.

low-level


It is capable of reading scancodes (keys), dealing with the text mode. I was able to turn on paging, protected 64-bit mode and execute Rust unsafe code (which runs even on barebone metal, without kernel libraries obviously). There is generated ISO image, which is enough to run inside QEMU + debugger (like gdb). I'm proud of it, but for those, who haven't experienced writing own OS firsthand, it may look crappy and straightforward. Even my lecturer wasn't 100% sure if it's feasable in Rust .
